# FFTrix: "Classic CV" Surveillance Platform

FFTrix is an enterprise-grade video surveillance and analytics engine. It is built on a "Classic CV" architecture—prioritizing deterministic, high-performance video processing using OpenCV and pattern-recognition algorithms instead of resource-intensive deep learning models.

### ⌨️ [CLI Reference](cli.md)

### 🔍 [Analytics Engine](analytics.md)
